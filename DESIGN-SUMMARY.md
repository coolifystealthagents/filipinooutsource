# Design summary

- Design concept: Warm recruiting field guide
- Layout personality: organic editorial
- Typography: Fraunces headings with Nunito Sans body text
- Color palette: primary #7c2d12, secondary #fbbf24, accent #16a34a, background #fffbeb, text #292524, muted #78716c
- Shape language: Large rounded cards
- Visual motif: passport stamps and warm paper blocks
- Unique layout choices: first sections use Talent field notes, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
